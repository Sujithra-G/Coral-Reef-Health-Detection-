# Coral Reef Health Detection using CNN

## 📌 Project Overview
This project focuses on detecting the health condition of coral reefs using Deep Learning techniques. A Convolutional Neural Network (CNN) is trained to classify coral images into three categories: Healthy, Bleached, and Dead.

## 🧠 Technologies Used
- Python
- TensorFlow & Keras
- CNN (Convolutional Neural Network)
- OpenCV
- Matplotlib

## 📂 Dataset Description
The dataset consists of underwater coral reef images categorized into:
- Healthy
- Bleached
- Dead

The dataset is split into training and testing sets.

## ⚙️ Methodology
1. Image preprocessing and normalization
2. CNN model training using TensorFlow
3. Model evaluation using accuracy and loss graphs
4. Testing model with unseen coral images

## 📊 Results
The trained CNN model achieved good accuracy in classifying coral reef health conditions. The model successfully predicts coral health on unseen images.
